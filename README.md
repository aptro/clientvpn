# ubuntu-vpn-client

## Setup
```
wget https://git.io/vFsp3 -O setup.sh && sudo sh setup.sh
```

## Start
```
wget https://git.io/vFshc -O start.sh && sudo sh start.sh
```

## Stop
```
wget https://git.io/vFshD -O stop.sh && sudo sh stop.sh
```
